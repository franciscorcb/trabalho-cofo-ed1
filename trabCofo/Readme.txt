------------------ Informações Técnicas ------------------ 

Sistema Operacional: Windons 11
Compilador: gcc
Linguagem: C 
IDE: Visual Studio Code
Como compilar: 
Primeiro colar esse codigo no terminal do Vscode
gcc apl.c matrix.c matrix.h -o teste.exe 
Logo após, executar o arquivo teste.exe na pasta

------------------ Sobre o Código ------------------ 

Arquivo apl.c:
Aqui possui minha função main e vai chamar as funções do meu TAD que foram implementadas 
no arquivo "matrix.c".

Arquivo matrix.h:
Nessa, possui as especificações do meu TAD, como tipos, definições, e o conjunto das minhas
funções que representam as operações. 

matrix.c:
Aqui fica a implementação das funções presentes no meu TAD em que usarei em no meu apl.c
